# EntertainmentBot 🎉

EntertainmentBot is a lightweight entertainment agent built on the **NemoBot platform**, designed to help users take a break and have fun.

It includes **5 main entertainment functions**, such as:

* Trivia questions
* Jokes
* Music suggestions
* Browser games
* Visual/interactive content

Whether you're bored from studying or just need a quick distraction, EntertainmentBot has something for you.
